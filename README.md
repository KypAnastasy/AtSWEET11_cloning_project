## **README (English)**

### Project Overview
Project: In Silico Design of Expression Vector for AtSWEET11 Gene
Project Overview

This repository contains the in silico design of a plant expression vector for constitutive expression of the AtSWEET11 gene under the CaMV 35S promoter with the NOS terminator. The vector was assembled using the Gateway MultiSite recombination system (4-1-2-3). All sequences were retrieved from NCBI, and Gateway adapters were used for recombination. The project aims to generate a stable expression system for functional studies of the AtSWEET11 gene, which encodes a sugar transporter involved in carbohydrate allocation.

Design Strategy

The vector assembly follows a four-fragment Gateway MultiSite cloning strategy (4-1-2-3), where the promoter, gene, and terminator are sequentially inserted into the final vector.

Vector Components

Promoter: CaMV 35S
The CaMV 35S promoter is a widely used constitutive promoter for plant expression, ensuring high and stable gene expression in most plant tissues.

Gene: AtSWEET11
The AtSWEET11 gene encodes a sugar transporter involved in the distribution of sugars throughout the plant. Its expression can be studied to analyze its function in sugar metabolism.

Terminator: NOS
The NOS terminator is used to ensure proper transcription termination and increase mRNA stability. It is commonly used in plant genetic constructs.

Cloning System

Gateway MultiSite (4-1-2-3):
The Gateway system allows for efficient modular assembly of genetic elements without the need for restriction enzymes and ligases. It provides a quick and accurate way to assemble complex constructs.

Entry Vectors
pDONR P4-P1R (Promoter)

The pDONR P4-P1R vector is used for promoter insertion into the construct. This vector contains attB adapters, which are necessary for recombination with the destination vector. The vector is ideally suited for the insertion of CaMV 35S, which is a widely used high-expression promoter in plant genetic engineering. The pDONR P4-P1R vector ensures efficient integration of the promoter, enabling robust expression in plant cells.

pDONR 221 (Gene)

The pDONR 221 vector is used for inserting the AtSWEET11 gene into the system. It contains attB adapters, enabling easy integration of the gene into other Gateway-compatible vectors. This vector is ideal for working with any gene of interest, making it a versatile tool for cloning applications. In this project, pDONR 221 was used to clone AtSWEET11, a gene encoding a sugar transporter involved in carbohydrate metabolism.

pDONR P2R-P3 (Terminator)

The pDONR P2R-P3 vector is designed for the insertion of the NOS terminator into the construct. It also contains attB adapters, allowing efficient integration of the terminator into the Gateway system. The NOS terminator ensures correct transcription termination and enhances the stability of the expressed mRNA, which is essential for stable gene expression in plant cells. This vector is particularly useful for cloning constructs with high-expression terminators like NOS.

Destination Vector
pDEST R4-R3

The pDEST R4-R3 vector is the destination vector where all genetic elements (promoter, gene, and terminator) are assembled. It supports the Gateway MultiSite system, which ensures the precise integration of all elements into the final expression construct. pDEST R4-R3 guarantees high recombination efficiency and stable expression in plants, making it an ideal choice for generating transgenic plants with the desired construct. The pDEST R4-R3 targeting vector contains ampicillin as a selectable marker for the selection of transformed cells.

Cloning System

Gateway MultiSite (4-1-2-3):
The Gateway system allows for efficient modular assembly of genetic elements without the need for restriction enzymes and ligases. It provides a quick and accurate way to assemble complex constructs. Recombination in the Gateway system is accomplished through the use of recombination enzymes called phage integrases.

## **README (русский)**

### Обзор проекта
Проект: Разработка in silico вектора экспрессии для гена AtSWEET11
Обзор проекта

В этом репозитории содержится разработка in silico вектора экспрессии для растений, обеспечивающего конститутивную экспрессию гена AtSWEET11 под промотором 35S вируса CaMV с терминатором NOS. Вектор был собран с использованием системы многосайтовой рекомбинации Gateway (4-1-2-3). Все последовательности были получены из NCBI, а для рекомбинации использовались адаптеры Gateway. Цель проекта — создание стабильной системы экспрессии для функциональных исследований гена AtSWEET11, кодирующего транспортер сахаров, участвующий в распределении углеводов.

Стратегия разработки

Сборка вектора осуществляется по четырёхфрагментной стратегии многосайтового клонирования Gateway (4-1-2-3), при которой промотор, ген и терминатор последовательно встраиваются в конечный вектор.

Компоненты вектора

Промотор: CaMV 35S
Промотор CaMV 35S — широко используемый конститутивный промотор для экспрессии генов в растениях, обеспечивающий высокую и стабильную экспрессию генов в большинстве растительных тканей.

Ген: AtSWEET11
Ген AtSWEET11 кодирует транспортер сахаров, участвующий в распределении сахаров по всему растению. Его экспрессия может быть изучена для анализа его роли в метаболизме сахаров.

Терминатор: NOS
Терминатор NOS используется для обеспечения правильной терминации транскрипции и повышения стабильности мРНК. Он широко используется в генетических конструкциях растений.

Система клонирования

Gateway MultiSite (4-1-2-3):
Система Gateway обеспечивает эффективную модульную сборку генетических элементов без использования рестриктаз и лигаз. Она обеспечивает быстрый и точный способ сборки сложных конструкций.

Векторы ввода
pDONR P4-P1R (Промотор)

Вектор pDONR P4-P1R используется для вставки промотора в конструкцию. Этот вектор содержит адаптеры attB, необходимые для рекомбинации с целевым вектором. Вектор идеально подходит для вставки промотора CaMV 35S, широко используемого высокоэкспрессируемого промотора в генной инженерии растений. Вектор pDONR P4-P1R обеспечивает эффективную интеграцию промотора, обеспечивая надежную экспрессию в растительных клетках.

pDONR 221 (Ген)

Вектор pDONR 221 используется для вставки гена AtSWEET11 в систему. Он содержит адаптеры attB, что обеспечивает легкую интеграцию гена в другие векторы, совместимые с Gateway. Этот вектор идеально подходит для работы с любым интересующим геном, что делает его универсальным инструментом для клонирования. В данном проекте pDONR 221 использовался для клонирования AtSWEET11, гена, кодирующего транспортер сахаров, участвующий в метаболизме углеводов.

pDONR P2R-P3 (Терминатор)

Вектор pDONR P2R-P3 предназначен для вставки терминатора NOS в конструкцию. Он также содержит адаптеры attB, обеспечивающие эффективную интеграцию терминатора в систему Gateway. Терминатор NOS обеспечивает корректную терминацию транскрипции и повышает стабильность экспрессируемой мРНК, что необходимо для стабильной экспрессии генов в растительных клетках. Этот вектор особенно полезен для клонирования конструкций с высокоэкспрессируемыми терминаторами, такими как NOS.

Целевой вектор
pDEST R4-R3

Вектор pDEST R4-R3 является целевым вектором, в котором собираются все генетические элементы (промотор, ген и терминатор). Он поддерживает систему Gateway MultiSite, которая обеспечивает точную интеграцию всех элементов в конечную экспрессионную конструкцию. pDEST R4-R3 гарантирует высокую эффективность рекомбинации и стабильную экспрессию в растениях, что делает его идеальным выбором для создания трансгенных растений с желаемой конструкцией. Вектор pDEST R4-R3 содержит ампициллин в качестве селективного маркера для отбора трансформированных клеток.

Система клонирования

Gateway MultiSite (4-1-2-3):
Система Gateway обеспечивает эффективную модульную сборку генетических элементов без использования рестриктаз и лигаз. Она обеспечивает быстрый и точный способ сборки сложных конструкций. Рекомбинация в системе Gateway осуществляется с помощью ферментов рекомбинации, называемых фаговыми интегразами.
